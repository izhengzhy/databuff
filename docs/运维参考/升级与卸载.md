# 升级与卸载

Docker 环境下的版本升级与卸载。

## 升级

已有环境换版本请用 **升级** 命令，不要用 `install.sh` 重装——重装会清空 `data/` 中的观测数据。

### 平台 · 在线升级

联网环境下，一条命令升级到最新版：

```bash
curl -fsSL https://databuff.ai/databuff/ai-apm-update.sh | bash
```

### 平台 · 离线升级

无法访问 CDN 时，下载目标版本离线包，解压后在目录内执行：

```bash
tar -zxvf databuff-ai-apm-offline-<version>-<arch>.tar.gz
cd databuff-ai-apm-offline-<version>-<arch>
sudo ./update.sh
```

离线包下载见 [离线安装](离线安装.md)。

### Demo · 在线升级

若已安装 Demo 应用，联网环境下执行：

```bash
curl -fsSL https://databuff.ai/databuff/ai-apm-demo-update.sh | bash
```

### Demo · 离线升级

在目标版本离线包解压目录内执行（会先停掉旧 Demo 再替换）：

```bash
sudo ./install_demo.sh
```

## 卸载

```bash
cd /opt/databuff-ai-apm
./stop.sh
cd ..
sudo rm -rf /opt/databuff-ai-apm
```

如需保留观测数据，卸载前备份 `data/` 目录。
