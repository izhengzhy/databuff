# HTML 风格参考模版

本目录下的 HTML **仅供参考色系与观感**，不是必选填空结构。  
在 `templates/` 下新增文件后，在本表追加一行即可；个数不限。

| 文件 | 大致方向 |
|------|----------|
| `summary-brief.html` | 短总结：结论优先、信息密度高、少装饰 |
| `report-analysis.html` | 长分析/报告：章节感、表格与证据区、卡片分区观感 |

读取路径（会话工作区）：

- 说明：`resources/skill.summary.html/templates/README.md`
- 模版：`resources/skill.summary.html/templates/{文件名}`
